# MailStub Site

Official website for [MailStub](https://github.com/jarednand/mailstub) - a lightweight email testing tool for developers.

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
